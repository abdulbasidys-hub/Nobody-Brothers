# Nobody Brothers — Maintenance Page

A one-page React + Vite site showing an "Under Maintenance" notice.

## Run locally

```bash
npm install
npm run dev
```

Open the URL Vite prints (usually http://localhost:5173).

## Build for production

```bash
npm run build
```

This outputs a static site into the `dist/` folder. Upload the **contents of `dist/`** to your host (or point Vercel/Netlify at this repo and let it run `npm run build` automatically — output directory `dist`).

## Preview the production build

```bash
npm run preview
```

## Where to edit

- Text and layout: `src/App.jsx`
- Colors, fonts, animations: `src/index.css`
- Page title / meta / favicon: `index.html`, `public/favicon.svg`
